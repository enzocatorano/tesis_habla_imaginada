# preproc_v0_test_rf_mlp

**Ejercicio de prueba**: estructura de experimento ficticio para ensayar la organización de pipelines.
**ATENCIÓN**: Esto es sólo un ejemplo para probar organización y reproducibilidad, **no** un experimento real.

Estructura:
- `data/` : .npz procesados específicos de este experimento
- `features/` : X.npy, y.npy resultantes
- `models/` : modelos entrenados
- `results/` : métricas, plots y logs

Config file: `config.yml` (parámetros de preprocesado, features y modelo).
